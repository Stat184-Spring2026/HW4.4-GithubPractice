# Reproducible Data Analysis Project

## What this project is about
This project is about doing data analysis in a clear and reproducible way using R and Quarto. It includes working with data, making graphs, and running a Monte Carlo simulation. The goal is to show all steps so someone else can run the code and get the same results.

## Where the data comes from
The airport data was created using passenger numbers for major airports from 2020 to 2025. The Monte Carlo part uses data that is generated in the code.
